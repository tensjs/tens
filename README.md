[![Greenkeeper badge](https://badges.greenkeeper.io/tensjs/tens.svg)](https://greenkeeper.io/)
Tens, written in Javascript and capable of running on top of TensorFlow.

# run

```shell
npm i
node scripts/install
npm run rebuild
```
